# XYZ BANK

## Objetivo
Esse projeto tem como objetivo consolidar o aprendizado realizado nas aulas de Testes de Sistema. Desenvolvendo uma automação de testes nos cenários previamente docummentados.

## Requisitos

- Python 3.11.7
- Selenium 4.27.1
- Pytest 8.4.1

## Instalação
1. Baixe e instale a versão específica do Python:
[Download Python 3.11.7](https://www.python.org/downloads/release/python-3117/) 

2. Instale as versões do pytest e seslenium  
```bash
pip install selenium==4.27.1 pytest==8.4.1
```
## Execução dos Testes
No terminal da sua IDE digite o comando  
```bash
pytest
```
